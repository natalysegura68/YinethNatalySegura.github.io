SQL Project
Covid 19

Covid 19 project is a data exploration that shows important information and conclusions about the 2020 pandemic. These include: the percentage of people that die if they get covid, percentage of total cases based on population per country, countries with highest percentage cases compare to their population, countries with highest death per population, continents with highest death per population, total numbers around the world and the total number of people who have been vaccinated to date.
This project was created with the dataset updated until Feb 5, 2024. The Data was taken from ourworldindata.org, in an excel format, where it was edited. I got two tables as a result and then they were imported to azure data studio SQL server as Covid_Deaths and Covid_Vaccinations.

Excel Project
Bikes Commute


This project is an analysis and visualization of bikes purchases in three different regions of the world (Europe,North America and Pacific) and the personal, social and economic factors that influence the decision to buy a bike or not.
Bike Project has four different worksheets: First one is Data, where you can find the dataset was used to do the analyzes and then create the dashboard. Second is Clean Data, the dataset after dropped duplicates, expand the marital status column and the gender column and ranged the age column into adolescent, middle age and old. The third worksheet contains Pivot Tables designed with the clean data sheet and the last one is Dashboard, a visualization of all the analysis.
Data was taken from Alex The Analyst YouTube Channel.

Tableau Project
Seattle AirBNB

This project is a visualization of the Airbnb activity in Seattle in 2016. The Dashboard was created with a dataset taken from Kaggle which contained three excel worksheets: Listings, including full descriptions and average review score. Reviews, including unique id for each reviewer and detailed comments. And Calendar, including listing id and the price and availability for that day.
In the final dashboard you can find four charts that illustrate the average price you need to pay in an Airbnb depending on the numbers of rooms. The weekly revenue for an Airbnb in Seattle in 2016. An average price of an Airbnb based on the type of property and, an average price of an Airbnb depending on the location of this.
